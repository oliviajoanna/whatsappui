### Project Structure
The lib directory is organized using a layer-first approach to ensure scalability and ease of database integration later:
```
lib/
├── core/
│   └── theme/          # AppColors (Modern Palette) and Theme data
├── data/               # dummy_data.dart for UI testing
├── models/             # Data models (Chat, Status, Channel)
├── screens/            # Feature-specific screens
└── widgets/            # Reusable UI components
```
---
### 🚀 Getting Started
Clone the repository:
```
git clone https://github.com/oliviajoanna/whatsapp_clone_uas.git
```
Install dependencies:

```
flutter pub get
```
Run the app:

```
flutter run
```
